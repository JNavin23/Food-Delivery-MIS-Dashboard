# 🍔 Food Delivery MIS Dashboard (Excel)

## 📊 Overview

This project showcases an interactive MIS dashboard built using Microsoft Excel to analyze food delivery operations across Indian cities.

## 🎯 Objectives

* Track key performance indicators (KPIs)
* Analyze revenue and order trends
* Evaluate delivery efficiency
* Understand customer payment behavior

## 🛠 Tools Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Slicers

## 📈 Key Features

* KPI tracking (Total Orders, Revenue, Average Order Value, Delivery Time)
* Peak hour analysis
* City-wise revenue performance
* Payment mode distribution
* Delivery performance analysis
* Interactive filtering using slicers

## 📊 Insights Generated

* Identified peak order hours for better operational planning
* Compared performance across different cities
* Analyzed customer payment preferences (UPI, Cash, Card, Wallet)
* Evaluated delivery time efficiency across locations


## 🚀 Conclusion

This dashboard helps in making data-driven decisions to improve delivery performance, optimize operations, and enhance customer experience.
